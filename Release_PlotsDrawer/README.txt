Axis limits - Минимум и максимум осей координат.
Current plot - График, с которым взаимодействуем в данный момент (изначально он не выбран).
Add plot - Добавление графика из файла. В файле обязательно должны быть колонки (разделённые пробелом) в следующем порядке JD R(1, al) RError. Пример файла находится в корне программы ( data.txt ).
Plots - Тут отображается список добавленных графиков. Нажмите на добавленный график в списке, чтобы взаимодействовать с ним (взаимодействовать в Current plot).
Log - Если в каком-то из шагов вы ошиблись, то Log даст Вам об этом знать. Файл с логами на протяжении всей сессии расположен в корне программы с названием log.txt.
